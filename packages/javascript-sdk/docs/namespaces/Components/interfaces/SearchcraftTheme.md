# Interface: SearchcraftTheme

This web component adds Searchcraft's built-in css theme to your page's <head> tag.
It does not render anything visible, its only function is to manage the css styles on the page.
## Usage
```html
<!-- index.html -->
<searchcraft-theme />
```
