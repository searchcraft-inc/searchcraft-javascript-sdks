import { Tabs, TabItem } from '@astrojs/starlight/components';

# searchcraft-filter-panel

This web component represents a series of filters that allows users to refine and control their search queries by applying various filter criteria.

## Import

<Tabs>
<TabItem label="React">
```jsx
import { SearchcraftFilterPanel } from "@searchcraft/react-sdk";
```
</TabItem>
<TabItem label="Vue">
```jsx
import { SearchcraftFilterPanel } from "@searchcraft/vue-sdk";
```
</TabItem>
</Tabs>


## Usage

<Tabs>
<TabItem label="JS">
```html
<!-- index.html -->
<searchcraft-filter-panel />
```

```js
// index.js
const filterPanel = document.querySelector('searchcraft-filter-panel');

if (filterPanel) {
  filterPanel.items = [];
}
```
</TabItem>
<TabItem label="React">
```jsx
<SearchcraftFilterPanel items={[]} />
```
</TabItem>
<TabItem label="Vue">
```jsx
<SearchcraftFilterPanel :items="[]" />
```
</TabItem>
</Tabs>


## Properties

| Property | Attribute | Description | Type | Default |
| -------- | --------- | ----------- | ---- | ------- |
| `items` | -- | The items to filter. | `FilterItem[]` | `[]` |
| `searchcraftId` | `searchcraft-id` | The id of the Searchcraft instance that this component should use. | `string \| undefined` | `undefined` |
